# pasos para configurar un proyecto reflex en python

📗Crear la carpeta del proyecto

`mkdir app_FutbolitoOnline`

📗Ingresar a la carpeta creada

`cd app_FutbolitoOnline`

📗Inicializar nuestro proyecto

`git init`

📗Crear nuestro entorno virtual

`python -m venv .venv`

📗Acceder y activar nuestro entorno virtual

`source .venv/Scripts/activate`

📗Para actualizar reflex

`python -m pip install --upgrade pip`

📗Instalamos reflex en nuestro proyecto

`pip install reflex`

📗Inicializamos reflex

`reflex init`

📗Para desactivar nuestro entorno virtual

`deactivate`

# Pasos para vincular un proyecto reflex con el repositorio git hub
👍Inicializar un repositorio git local

`git init`

👍Agrega los archivos a git

`git add .`

👍Vizualisar archivos que estamos guardando

`git status`

👍Comentar los cambios

`git commit -m "primer commit en proyecto reflex"`

👍Vincular el repositorio local con el repositorio remoto en github

`git remote add origin <url_del_repositorio>`

👍 Para subir cambios

 `git push -u origin main`
